## 📦 Tools

- **[DarkRadiant](https://www.darkradiant.net/)** – Modern level editor for idTech4-based games.
- **[NetRadiant-Custom](https://netradiant.gitlab.io/)** – Another modern lever editor, not has feature rich has DarkRadiant, but nice to have.
- **[MD5 Blender Export Script](https://github.com/KozGit/Blender-2.8-MD5-import-export-addon)** – Community script for exporting models to MD5 format **DOES NOT WORK IN BLENDER 4.X**.
- **[Aseprite/LWO Blender Export Scripts](https://gitlab.com/orbweaver/darkblender)** – Community script for exporting models to ASE/LWO formats **DOES NOT WORK IN BLENDER 4.X**.
- **[Lightwave to MD5 converter](https://www.moddb.com/games/doom-iii/downloads/lightwave-to-md5-converter)** - This is a utility that converts LWO/LWS model files into MD5 files.
- **[Doom 3 - Quake 3 Map Converter](https://www.moddb.com/games/doom-iii/downloads/doom-3-quake-3-map-converter)** - Use this tool to convert your favorite Quake 3 maps to Doom 3's (Quake 4 also) new format.
- **[ROQ Creator](https://www.moddb.com/games/doom-iii/downloads/roq-creator)** - Convert your TGA sequence into an ROQ video file for use within the Quake 4 and Doom 3 games.
- **[Quake Video Maker](https://www.moddb.com/games/doom-iii/downloads/quake-video-maker-142)** - This is a custom utility that allows you to create custom .roq files for Quake 3 and Doom 3.
- **[MD5 Model Viewer](https://www.moddb.com/games/doom-iii/downloads/md5-modelviewer)** - This is a custom made model viewer for MD5 meshes. Created by der_ton and originally posted to Doom3World.
- **[Doom Maps Converter](https://www.moddb.com/games/doom-iii/downloads/doom-maps-converter-14)** - Converter of old Doom maps to maps for Doom 3, made from QuakeDM sources.
- **[maGen Material Generator](https://www.moddb.com/games/doom-iii/downloads/magen-material-generator)** - This tool is for batch generation of material files for DOOM3.
- **[Doom 3 BFG FontTools](https://www.moddb.com/games/doom-iii/downloads/doom-3-bfg-fonttools-155)** - 1-click font creation software for Doom 3 BFG Edition. Make game-ready fonts from .otf and .ttf fonts.
- **[BFG Resource File Manager](https://github.com/MadDeCoDeR/BFG-Resource-File-Manager)** - Open and edit DOOM 3 BFG/2019 .resources files

## 📚 Resources

- **[Idtech 4 Ever](https://discord.gg/tJDGrk6w4H)** - A great server dedicate to idtech4.

- **[ModWiki](https://modwiki.dhewm3.org/)** – Comprehensive wiki for idTech4 games.

- **[Quake 4 Wiki](https://iddevnet.dhewm3.org/quake4/Quake4SDK.html)** - Some tutorials in here are missing in modwiki.

  - **[Doom 3 BFG Source Code](https://github.com/id-Software/DOOM-3-BFG)** – Doom 3 GPL source code.
- **[Doom3World.org (archived)](https://web.archive.org/web/20040801000000*/Doom3World.org)** – Good old Doom3World, while dead, some of its pages are archive.
- **[DoomWorld Doom 3 Section](https://www.doomworld.com/forum/8-doom-3/)** - Doom 3 Section of DoomWorld, really good resource of learning.
- **[Technical Papers](https://github.com/FriskTheFallenHuman/awesome-idtech4/tree/main/docs)** – MrElusive and Carmack’s papers about the engine.
  - **[Fabien Notes](https://fabiensanglard.net/doom3/doom3_notes.txt)** - A great review in how the engine works.
  - **[Fabien Doom 3 Review](https://fabiensanglard.net/doom3/)** - Paired with fabian notes, the Doom 3 Source review its also a great learning source.
  - **[Fabien Doom 3 BFG Review](https://fabiensanglard.net/doom3_bfg/)** - Same as above but for BFG Edition.
- **[DoomEdit Ultimate Tutorial](https://www.moddb.com/games/doom-iii/downloads/doomedit-ultimate-tutorial)** - The Ultimate DOOMEdit Tutorial since doom3world.org is down made by me.
- **[Doom 3 Reference Page](https://www.moddb.com/games/doom-iii/downloads/doom-3-reference-page)** - The Doom 3 Reference Page is a Windows help file that contains information on Doom 3's CVar's, script commands and command line arguments.
- **[DOOMSCRIPT Highlighter Notepad++](https://www.moddb.com/games/doom-iii/downloads/doom3-script-highlighter-notepad)** - Detailed addon for DOOMSCRIPT(DOOM 3) with complete highlighting, autocompletion and a custom theme for the Notepad++. Entity, mover, AI and System events all have differing colors to make them easily identifiable.
- **[Maxell's Tutorials for id Tech 4 engine](https://steamcommunity.com/sharedfiles/filedetails/?l=polish&id=2775368502)** - This is the complete translation of the tutorial made by Martin Pezlar aka Maxell that will help to understand and modify the id tech 4 engine, keep in mind that this is the "old" way meaning that most of the tools and procedures described are around the time the game was released.
- **[Moddb Doom 3 Tutorials](https://www.moddb.com/games/doom-iii/tutorials)** - A great list of tutorials for Doom 3
- **[3D Buzz Doom 3 Tutorials: Series](https://www.youtube.com/watch?v=IDN5Od0z2CY&list=PLzlHQaNpSPoWgYvINqDSsrepL_-Rq6eya&index=1)** - Great tutorial from 3D Buzz covering doom 3 mapping
- **[Doom3World Mapping Tutorial Series](https://www.youtube.com/watch?v=nwGBstvlM2A&list=PLzlHQaNpSPoXgQeDdjub4EbUIZbYPOftm)** - This is the first in a series of Doom 3 video tutorials created by Stephen Rosebrough for the website Doom3world.org.
- **[Doom 3 Mapping 101](https://www.youtube.com/watch?v=ow6V2Ju5hWw&list=PLzlHQaNpSPoWYLVZy6K36wNOK73FHm4Zm)** - Another great series from Doom3World.
- **[Doom 3 Weapon Creation Tutorial](https://www.youtube.com/watch?v=M2DKEnZIAPk&list=PLzlHQaNpSPoU-Z3zoVhauffXPttJlcuUm)** - This tutorial series goes over how to replace the pistol model and animation with a different one using 3dsmax.
- **[Quake 4 Modding Tutorial](https://www.youtube.com/watch?v=rmMjWSlbOow&list=PLzlHQaNpSPoUYtIQdDqxkqhanF8H5mXJ6)** - Tutorial originally created by foyleman.
- **[Prey Mapping Tutorial](https://www.youtube.com/watch?v=UCVuV3Jb8K0&list=PLzlHQaNpSPoW18X2j8D-qX8AJ8pTDQHYm)** - Tutorial originally created by foyleman.

## 🎨 Assets

- **[BlendoGames Starter Kit](https://github.com/blendogames/Doom3-Starter-Kit)** - Starter kit for standalone Doom 3 projects.
- **[D3 Base Assets](https://github.com/DanielGibson/d3-base-assets)** - This aims to become a base for standalone games based on the free Doom3 engine.
- **[Basic Standalone pack for Game Developers ](https://www.moddb.com/games/doom-iii/downloads/basic-standalone-pack-for-game-developers-alpha-05)** - This is a pack with the very basic assets for Indie game projects using the GPL Doom 3 (2004) engine (or any port)
- **[Standalone Game Asset & source Pack](https://www.moddb.com/games/doom-iii/downloads/standalone-game-asset-and-source-pack)** - Assets and their sources for gamedevs using idTech 4 - maps, models and textures.
- **[Doom 3 Can Do It Too](https://www.moddb.com/games/doom-iii/downloads/doom-3-can-do-it-too-code-mods)** - This is a pack of mods that implement 3D skyboxes, water and bloom into Doom 3. Also included are their example files.
- **[Mapper's Survival Kit](https://www.moddb.com/games/doom-iii/downloads/mappers-survival-kit1)** - This file includes 7 packs full of textures that you can use freely in your projects.
- **[CC0 Skybox Pack #1](https://www.moddb.com/games/doom-iii/addons/cc0-skybox-pack-1)** - A tiny pack of skybox textures released under CC0 license.
- **[CC0 Skybox Pack #2](https://www.moddb.com/games/doom-iii/addons/cc0-skybox-pack-2)** - A second tiny pack of skybox textures released under CC0 license (the first one is in addons to Doom 3)
- **[OpenSource Art](https://opengameart.org/)** - Great site with CC0 models/textures and sounds.
- **[LibreQuake Assets](https://github.com/lavenderdotpet/LibreQuake)** - LibreQuake assets are on the CC0 license.
